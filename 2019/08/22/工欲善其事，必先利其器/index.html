Typora 和MarkDown入门

1. typora

什么是typora

​ typora 是一款专门用于编写markdown的书写工具，功能十分强大，能够让你像书写word一般，丝滑流畅的编写markdown文档，从而不用去纠结其中语法的细节。

​ 具体详细的介绍，我就不多说了，百度上一搜一大把，我想写的是，能够快速的了解这个软件是什么，如何使用其中的基础功能，达到这一个目的就行。因为你刚接触一个自己以前从来没有接触过的东西的时候，陌生感和畏难情绪是最需要先克服的，其他的东西，在你不断的接触和深入过程中，你慢慢的就能了解了。

简单使用typora

​ 下载安装就是傻瓜式操作就行，无脑下一步，之后打开typora。它的语言默认与系统语言相一致，一定要先自己对功能栏每一个都读一遍，大概了解它都有那些功能，这一步是最关键的，其中很多东西你可能暂时还不明白，但是一定要先看一下，自己有一个了解。

​ 看过之后，新建一个新的文档，通过菜单栏可以对你写入的文字进行格式的修改，也可以通过快捷键来实现。一切的基本使用都是和word十分相似的，会word，肯定能很快上手typora。

​ 具体的一些内容分述如下：

​ 常用快捷键

  • 加粗： Ctrl/Cmd + B
  • 标题： Ctrl/Cmd + H
  • 插入链接： Ctrl/Cmd + K
  • 插入代码： Ctrl/Cmd + Shift + C
  • 行内代码： Ctrl/Cmd + Shift + K
  • 插入图片： Ctrl/Cmd + Shift + I
  • 无序列表： Ctrl/Cmd + Shift + L
  • 撤销： Ctrl/Cmd + Z
  • 一级标题：快捷键为Crtl + 1，以此类推

2. Markdown

简介

​ Markdown是一种标记语言，它具有简单的标记语法，通过这些标记语法，它可以使普通文本内容具有一定的格式。

​ markdown具有两大特点：易写，易读

​ 「易写」不需要用鼠标，通过几个字符就能清楚地设定文章结构，进行文章排版，而且语法极其简单，学习成本非常低，可以轻松的进入沉浸式写作，而不用纠结于排版。

​ 「易读」Markdown具有极强的兼容性，转换为富文本或者HTML之后还能保持原有的格式。不会出现我是用pages写的，windows打不开，或者我是新版的word写的，旧版的打开后格式有问题这类情况。Markdown具有极强的兼容性，转换为富文本或者HTML之后还能保持原有的格式。不会出现我是用pages写的，windows打不开，或者我是新版的word写的，旧版的打开后格式有问题这类情况。

一些语法内容

​ 注意，在typora中，即写即所得，所以这些语法也不是非要记住不可，使用工具，可以让自己的效率提高不少！

换行符

在markdown中，段落由多个空格分隔。在Typora中，只需回车即可创建新段落。

标题级别

# 一级标题， 快捷键为Crtl + 1
## 二级标题，快捷键为Crtl + 2
###### 六级标题，快捷键为Crtl + 6

引用文字

> + 空格 + 引用文字

清单

输入 列表内容 将创建一个无序列表，该符号可以替换为+或-。

输入1. 列表内容 将创建一个有序列表，其markdown源代码如下：

无序列表
* AA
* BB
* CC

有序列表

  1. AA
  2. BB
  3. CC

任务列表

- [ ] 不勾选
- [x] 勾选
